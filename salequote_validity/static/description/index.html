Sale Expiration Date Reminder

Set your Sale Expiration Reminder

Features

You can run reminder manually

Set the reminder to the quotation/order

Remind the expiration automatically

Step 01

This odoo module enhances sales features. You have to first set the above odoo module.Then you can install the Sale expiration date reminder module.

Step 02

Goto, Main menu > Sales > Create Quotation

Here you can set the quotation/order expiration date.

Step 03

If you want to run scheduler manually,

You can get that using Settings > Technical > Scheduled action.

Step 04

After get the scheduled action, In that section, you can select the validity due date reminder.

Step 05

Here you can see all the details about the scheduler and then you can run the scheduler manually. Then system will automatically send an email to related partners.

Step 06

You can check the mails in the mails section.
